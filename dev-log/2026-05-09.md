## 部署步骤（待用户操作）
1. 在 GitHub 创建新仓库
2. 推送：`git remote add origin <repo-url> && git push -u origin main`
3. Settings → Pages → Source: Deploy from main branch, folder: /product-eye (或 /)
4. 手机 Chrome 打开 `https://<user>.github.io/<repo>/` 或 `https://<user>.github.io/<repo>/product-eye/`
5. 添加到桌面：Chrome 菜单 → 添加到主屏幕
